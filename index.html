HTML

  • div creates a division in the page
  • span creates an inline division in the page
  • p represents a block of text
  • ul, ol, li represent unirdered, ordered, and list items
  • img contains information about images and contains src attribute
  • the a element stands for anchor and creates links to webpages
  • href attributes points to the url link
  • title is important for SEO
  • There are 6 types of headers- h1 being the largest

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • an array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • A for loop tells computer to go through each shape in our array and log them into console until none left
  • Array length is important because for loop needs to know how many times code will run until no items left in array
  • For loop has a starting point, a condition the loop will run on, and what allows loop to reiterate
